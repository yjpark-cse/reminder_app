# 건강관리 리마인더 앱

Flutter와 Firebase를 활용한 개인 맞춤형 건강관리 앱입니다.  
물 섭취, 약 복용 등의 정보를 기록하고 홈 위젯과 연동하여 간편하게 관리할 수 있습니다.

## 주요 기능
- 음식 및 칼로리 기록
- 하루 물 섭취량 체크
- 영양제/약 복용 체크
- 캘린더 기반 섭취 기록
- 홈 화면 위젯 연동
- Firebase 연동 (Firestore 사용)

## 사용 기술
- Flutter / Dart
- Firebase (Authentication, Firestore)
- SharedPreferences
- Android Studio

## 개발 목적
학술제 출품용 개인 프로젝트로, Flutter 기반 앱 개발 역량을 기르고  
Firebase 및 위젯 연동에 대한 실무적 경험을 쌓기 위해 제작하였습니다.
