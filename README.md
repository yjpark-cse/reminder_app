# 건강관리 리마인더 앱 (개발 진행 중)

Flutter와 Firebase를 활용하여 개인 맞춤형 건강 관리를 돕는 리마인더 앱입니다.
하루 식단, 수분 섭취, 약 복용 등의 건강 정보를 간편하게 기록하고,
홈 화면 위젯을 통해 빠르게 체크할 수 있도록 개발 중입니다.

## 주요 기능 (개발 중 / 일부 구현 완료)
- 하루 물 섭취량 기록 및 목표 설정
- 식단 및 칼로리 기록
- 영양제/약 복용 체크
- 캘린더 기반 건강 기록 시각화
- 홈 화면 위젯 연동 (물 섭취 버튼, 약 복용 체크)
- Firebase 연동 (Firestore 사용)

## 사용 기술
- Flutter / Dart
- Firebase (Authentication, Firestore)
- SharedPreferences
- Android Studio

## 개발 목적
2025년 학술제 출품용 개인 프로젝트로, Flutter 기반 앱 개발 역량을 기르고  
Firebase 및 위젯 연동에 대한 실무적 경험을 쌓기 위해 기획되었습니다.
